# assets/

Put images, screenshots, demo GIFs, and cover illustrations here.

**Suggested files:**
- `cover.png`: the banner shown at the top of the README
- `home.png`, `flow.png`, ...: screenshots referenced from the README
- `demo.gif`: a short (under 15s) screen recording of the key flow

**Tips**
- Keep file sizes reasonable. Compress PNGs, keep GIFs under a few MB.
- For longer videos, upload to Drive/YouTube and link from the README instead of committing.
- Reference these files from `README.md` using relative paths, e.g. `![home](assets/home.png)`.
